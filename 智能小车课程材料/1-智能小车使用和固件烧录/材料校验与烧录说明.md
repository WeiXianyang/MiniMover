## 4. 底盘固件烧录流程

以下流程根据手册第 3.12 节与材料包内容整理：

1. 在笔记本电脑上准备 `ArteryISPProgrammer` 工具。
2. 将底盘控制线的 USB 一端插入运行 `ArteryISPProgrammer` 的笔记本 USB 接口。
3. 将小车底盘的 `BOOT1` 按钮拨到上方 `VDD`。
4. 打开烧录软件并按向导进入文件选择界面。
5. 如已有旧烧录文件，先删除旧条目。
6. 点击添加，选择目标固件：
   - 循线/IOT：`底盘固件包/follow-line.bin`
   - 常规 ROS：`底盘固件包/iCar.bin`
7. 确认后继续执行擦除、下载、烧写。
8. 烧写完成后，将 `BOOT1` 按钮拨回下方 `GND`。
9. 将底盘控制线 USB 端插回小车顶部中间 USB 接口。
10. 上电或重启后进行对应功能验证。

## 5. 循线验证流程

1. 烧录 `follow-line.bin`。
2. 通过 VNC Viewer 远程连接小车系统。
3. 在 Terminal 中依次输入：

```bash
ros
app
```

4. 将小车放置在循线地图上。
5. 在 APP 界面左下角勾选“循迹指令”。
6. 小车应沿黑线运动；停止时可抱起小车并取消勾选“循迹指令”。
7. 循线测试结束后，如需做雷达/导航/APP 常规控制实验，刷回 `iCar.bin`。

## 6. 常规功能验证建议

烧录 `iCar.bin` 后，建议至少验证以下一项常规功能：

| 功能 | 参考手册章节 | 验证要点 |
| --- | --- | --- |
| APP 控制 | 3.5 或 3.6 | 小车可前进、后退、平移、转向、停止。 |
| SLAM 建图 | 3.7 | RViz 可显示激光扫描，键盘控制可完成建图并保存地图。 |
| 自动导航 | 3.7 | 设置初始位姿和目标点后，小车可规划并移动到目标点。 |
| 雷达避障 | 3.8 | 小车自主运动时可规避前方障碍物。 |
| 雷达跟随/警卫 | 3.9/3.10 | 小车能跟随目标或保持朝向目标。 |
