# 七、巡检业务流程图（Mermaid）

## 用法：展示从发起到归档的完整业务闭环

```mermaid
sequenceDiagram
    actor 用户
    participant APP as 鸿蒙APP
    participant 车端 as 车端API Server
    participant ROS2 as ROS2 Docker
    participant AI as AI识别模块
    participant 云 as 云平台

    用户->>APP: 点击"开始巡检"
    APP->>车端: POST /api/start_patrol
    车端->>ROS2: ros2 action send_goal
    ROS2-->>车端: 导航中...
    
    loop 每个巡检点
        ROS2->>ROS2: 激光雷达避障
        ROS2-->>车端: 到达巡检点
        车端->>AI: 触发视觉检测
        AI->>AI: YOLOv5火焰/烟雾识别
        AI->>AI: 多传感器融合判定
        
        alt 发现异常
            AI-->>车端: 高等级告警
            车端->>车端: 停车+抓拍
            车端->>云: POST /api/v1/fire-alarms
            云->>APP: WebSocket推送告警
            APP->>用户: 告警通知+截图
        else 正常
            AI-->>车端: 正常
            车端->>ROS2: 前往下一巡检点
        end
    end

    车端->>云: 巡检报告归档
    云->>APP: 报告可查看
    APP->>用户: 巡检完成+报告
```

# 附录：图表制作指南

## 各图表在PPT中的制作步骤

### 1. 雷达图（第六节 → 竞品对比）
1. Excel中粘贴"一、竞品雷达图数据"
2. 选中数据 → 插入 → 雷达图 → 填充雷达图
3. 6个维度各一条轴，6条不同颜色的线代表6个产品
4. 添加图例和数据标签

### 2. 散点图（第六节 → 竞品定位）
1. Excel中粘贴"二、竞品定位图数据"
2. 选中X/Y列 → 插入 → 散点图
3. 添加数据标签显示产品名
4. 注意突出FireGuard（大号星星标记）

### 3. 热力图（技术成熟度）
1. PPT中插入表格，填入"三、技术成熟度热力图"数据
2. 使用条件格式：0%红色 → 50%黄色 → 100%绿色
3. 或手动设置单元格底色

### 4. 瀑布图（差距分析）
1. Excel中插入 → 瀑布图
2. 数据为"四、差距分析瀑布图"
3. 显示总柱子从当前到目标的变化

### 5. Mermaid图（架构图/流程图）
1. 复制"六"或"七"节的Mermaid代码
2. 在 https://mermaid.live 粘贴 → 导出PNG/SVG
3. 将导出的图片插入PPT

### 6. 甘特图（里程碑）
1. Excel中插入堆积条形图
2. 使用"八、项目里程碑甘特图数据"
3. 或直接在PPT中手绘时间轴
