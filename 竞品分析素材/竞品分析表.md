### 竞品 1：亿嘉和 SI300 智能简易巡检机器人

#### 产品图片

#### 产品介绍
SI300 是亿嘉和面向配电房、开关室等场景推出的智能简易巡检机器人。采用差速驱动轮式底盘，支持 SLAM 实时建图与自主导航，可灵活搭载可见光、红外、气体检测等多种传感器。通过专业化技术规范设计，将巡视耗时从人工 4 小时/站压缩至 30 分钟/站以内，缺陷识别率提升至 99%。

|------|------|
| 尺寸 | φ480 × 1850mm |
| 重量 | <40kg |
| 防护等级 | IP4X |
| 运动速度 | 1m/s |
| 可见光相机 | 1600万/800万像素 |
| 红外成像 | 选配 |
| 建图定位精度 | ≤±20mm |
| 工作温度 | -5℃~45℃ |

**★★★★★（高度相似）** — SI300 是当前市场上与 FireGuard 定位最接近的产品。两者面向完全相同的场景（配电房巡检），采用相同的技术路线（SLAM导航+视觉识别+传感器监测），甚至产品形态（轮式机器人）和尺寸也相近。主要差异在于：SI300是商用成熟产品（IP4X防护、-5℃~45℃工作范围），FireGuard是教育/验证平台（更侧重功能演示和全栈集成）。FireGuard相比SI300的差异化优势在于：(1) LLM语音交互；(2) 多车协同框架；(3) 全开源技术栈。

- **间隙测量**：自动测量关键部件间隙
