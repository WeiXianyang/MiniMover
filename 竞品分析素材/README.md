## 使用方法

### 在需求分析报告中引用

**第3.3节 技术摸底**：将 `技术摸底表.md` 中的三张表格（已有能力评估、关键缺口、课程对照）复制到需求分析报告中，替换原有的"暂时无法在飞书文档外展示此内容"占位符。

**第3.4节 竞品分析**：将 `竞品分析表.md` 中的"竞品总览表"和"竞品矩阵总对比"复制到需求分析报告中，并按需引用各竞品的详细分析。

### 获取图片

参见 `图片下载指南.md`，从各官方产品页面截图或下载产品渲染图。
